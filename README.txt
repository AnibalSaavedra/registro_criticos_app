Registro de Notificación de Valores Críticos

Este proyecto permite registrar y gestionar las notificaciones de valores críticos de laboratorio en un entorno seguro, persistente y exportable.

Valores críticos considerados:
- INR ≥ 3.5 %
- Hematocrito ≤ 16 %
- Glucosa ≤ 40 mg/dl
- Potasio ≥ 6.2 mEq/L
- Troponina ≥ 0.12 ng/mL
