# Registro de Notificación de Valores Críticos

Esta aplicación en Streamlit permite registrar y gestionar notificaciones de valores críticos de laboratorio de forma automática y persistente.

### Valores críticos considerados:
- INR ≥ 3.5%
- Hematocrito ≤ 16%
- Glucosa ≤ 40 mg/dL
- Potasio ≥ 6.2 mEq/L
- Troponina ≥ 0.12 ng/mL

Todos los registros se almacenan automáticamente en un archivo Excel local llamado `registro_valores_criticos.xlsx`.
